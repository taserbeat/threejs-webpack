# threejs-webpack

[Three.js](https://threejs.org/)を webpack で実行、ビルドできるようにしたプロジェクト

# 要件

| 名称    | バージョン   |
| ------- | ------------ |
| Node.js | 16.14.0 以上 |
| yarn    | 1.22.10 以上 |

# クイックスタート

- 開発用ローカルサーバーを起動

  ```bash
  yarn start
  ```

- ビルド

  ```bash
  yarn build
  ```

- 開発用ローカルサーバーで一時的なホスティング

  ```bash
  yarn serve
  ```
